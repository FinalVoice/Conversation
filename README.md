オレとペットの非日常
==================
ペットの「ペチ」と会話する選択方式のプログラムです。  
ストーリーを進めてまともなエンディングに辿り着こう！(全８種)  
動作環境：Firefox


### 操作の仕方 ###
------
名前を入力し、ＯＫを押すとストーリーが始まります。  
プロローグが流れた後に会話が可能になります。  
選択肢を選んでストーリーを進めてください。  
やり直したい場合は「最初から」を押してください。
